# /capture Capture image

$ARGUMENTS

---

## Purpose

This command activates capture mode for how to using browser to capture image in current project. It full flow for access and login to capture image
---

## Steps
1. Access to URL: $url
2. Click on button: Login with root
3. Input $username and $password to input and press login
4. after login success you can redirect to another screen by click on sidebar menu(not use change url path)
5. Doing capture as request needed
